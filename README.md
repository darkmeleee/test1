# Create T3 App

Современный full-stack приложение для каталога цветов на T3 Stack с авторизацией через Telegram WebApp.

## Функциональность

- **Каталог цветов** с красивым grid-отображением
- **Управление категориями** - добавление основных категорий и атрибутов
- **Корзина** с подсчетом общей стоимости
- **Профиль пользователя** с данными из Telegram
- **Адаптивный дизайн** для мобильных устройств и Telegram Mini Apps
- **Next.js + TypeScript + tRPC + Prisma** для полной типобезопасности
- **Tailwind CSS** для стилизации
- **Telegram WebApp API** интеграция с авторизацией
- **Haptic feedback** для тактильных ощущений
- **Темная тема** поддержка

## Технологии

- **Frontend**: Next.js 14, React 18, TypeScript, Tailwind CSS
- **Backend**: tRPC, Prisma, SQLite
- **Authentication**: Telegram WebApp Login Widget
- **Icons**: Lucide React
